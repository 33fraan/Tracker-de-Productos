¿Qué tipo de publicación usar?
Premium (17%) — aparecés en los primeros resultados, incluye envío gratis para el comprador y cuotas sin interés. Obligatorio para escalar.
Clásica (13%) — menor visibilidad, sin envío gratis automático. Sirve para testear antes de escalar.
Gratuita (0%) — sin comisión pero con límite de unidades vendidas. Solo para arrancar.

☁️ Conectar Supabase
1Creá una cuenta gratuita en supabase.com → New Project
2En el SQL Editor de tu proyecto, ejecutá este query:
create table productos ( id bigserial primary key, nombre text, categoria text, estado text, proveedor text, web text, mayor numeric, moq integer, ml numeric, comision numeric, iva numeric, envio numeric, iibb numeric, competencia text, anmat text, notas text, created_at timestamptz default now() ); alter table productos enable row level security; create policy "open" on productos for all using (true) with check (true);
3Andá a Project Settings → API y copiá la Project URL y la anon public key
